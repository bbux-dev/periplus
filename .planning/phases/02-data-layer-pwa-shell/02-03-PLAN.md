| T-02-SW-01 | Tampering / Spoofing | generated service worker | mitigate | SW is generated from local source by Workbox `generateSW` (not fetched from an external URL); same-origin scope `'/'`; precache manifest uses content-hash revisions so tampered assets are detected. No external CDN scripts cached. |
| T-02-SW-02 | Information Disclosure | SW precache scope | accept | `globPatterns` precache only static app-shell assets (js/css/html/ico/png/svg/woff2).
