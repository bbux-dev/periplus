| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-23-06 | Spoofing | unstamped entry when no trip | mitigate | settled-signal guard redirects to '/create-trip' before save; draftToEntry only stamps tripId when activeMode.mode is truthy |
| T-23-07 | Tampering | unknown/forged :type slug | mitigate | ACTIVITY_TYPES.find(...) ?? 'Other' falls back safely; isOther forces a required free-text Type |
| T-23-08 | Information Disclosure | free-text rendered later | accept | values are React-escaped on render (no dangerouslySetInnerHTML in this flow); local single-user store |

No package installs in this plan (zero new deps), so no supply-chain (T-23-SC) checkpoint required.
