Structure:
- `const { pathname } = useLocation()`, `const navigate = useNavigate()`, `const [open, setOpen] = useState(false)`, and a per-domain expansion state (e.g. `const [expanded, setExpanded] = useState<Record<string, boolean>>({})`).

Sticky (in normal flow), NOT fixed ‚Äî pages keep their `min-h-screen` layouts and content flows below the bar without being occluded (locked decision 1). Inner row: `w-full max-w-sm mx-auto flex items-center justify-between px-6 h-14`.
- LEFT ‚Äî home button (locked decision 2): only render when `pathname !== '/'`. A `<button aria-label="Go home" onClick={() => navigate('/')}>` containing `HomeIcon` from `@heroicons/react/24/outline`, styled with `text-[var(--color-primary)]`.
- RIGHT ‚Äî hamburger (locked decision 3): `<button>` with `Bars3Icon` from `@heroicons/react/24/outline`, `aria-label="Toggle navigation menu"`, `aria-expanded={open}`, `aria-controls="app-nav-menu"`, `onClick={() => setOpen(v => !v)}`.
- DROPDOWN: when `open`, render `<nav id="app-nav-menu">` positioned below the bar (e.g. `absolute` panel within a `relative` wrapper, or a block under the header ‚Äî keep it simple and visible). Contents:

  - Nav tree from `NAVIGATION` (import it; do NOT hardcode domains/types ‚Äî locked decision + constraint): for each domain render a row with (a) a `<Link to={`/d/${domain}`}>` for the domain label that closes the menu on click, and (b) a separate expander `<button aria-label={`Expand ${label}`} aria-expanded={!!expanded[domain]} onClick={() => toggle expanded[domain]}>` with a `ChevronDownIcon`. When that domain is expanded, render its `types` as `<Link to={`/d/${domain}/${type}`}>` items (label = type config label) that also close the menu on click.

  - Escape: `useEffect` that, while `open`, adds a `document` `keydown` listener closing on `event.key === 'Escape'`; clean up on unmount/close.
  - Outside click: `useEffect` that, while `open`, adds a `document` `mousedown` listener; keep a `ref` on the wrapper containing the hamburger + panel and call `setOpen(false)` when the event target is outside it; clean up on unmount/close.
- After the header, render `{children}` (the routed pages).
- Use `cn()` from `../ui/cn` for className composition; mark decorative icons `aria-hidden="true"`.

Do NOT reduce scope (no "v1"/"static" placeholders) ‚Äî the full tree and all four top-level links ship in this task.
