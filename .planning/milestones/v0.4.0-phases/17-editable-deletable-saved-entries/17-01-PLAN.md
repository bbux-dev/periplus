<objective>
Make saved entries editable and deletable. Add an edit route `/entries/:id/edit` whose form
reuses `ENTRY_FIELDS[type]` + `buildReviewDraft`, pre-populated from the saved entry, persisting
through the EXISTING `entriesRepository.update`; add Edit + Delete (inline confirm) affordances to
`EntryDetailPage`, deleting through the existing `entriesRepository.delete`. `recordedAt` immutable;
`occurredAt` editable. Metadata is MERGED on save so unknown keys (URL/DSL capture; the upcoming
Phase 18 `mode`/`modeLabel` stamps) survive — and any such key is itself editable via an
"extra metadata" section (this is how mode re-assignment "falls out of metadata editing").
</objective>

      else `delete meta[key]`; then for each [key,raw] in extraMetadata, `raw.trim()` truthy →
      meta[key]=trimmed, else `delete meta[key]`. changes.metadata = meta. NEVER set recordedAt,
      syncedAt, domain, or type.
