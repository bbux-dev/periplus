## Skipped Issues

### IN-02: `short_name` duplicates `name` in the web manifest

**File:** `src/pwa/pwaConfig.ts:14`
**Reason:** No code change warranted — intentionally left as-is.
**Reasoning:** `short_name: 'Life Log'` is 8 characters, well within the 12-character Android home-screen guideline. Changing it to `'LifeLog'` (no space) saves one character but creates a branding inconsistency with `APP_NAME` and degrades readability. Omitting `short_name` entirely would cause browsers to fall back to `name` — the same value — producing no functional difference. The current state is correct and deliberate.
